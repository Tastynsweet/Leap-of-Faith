# Leap of Faith

A simple clicker game focused on incremental progression and score accumulation

## Overview
Leap of Faith is a casual clicker game where the player repeatedly clicks to gain faith and increase their total score over time. Its a game about depression and simple ways to deal with it. The project is technically complete and fully playable, with all core mechanics implemented.

This game was created as a small, self-contained project to practice gameplay logic, UI handling, and state management.

## Features
- Click-based score system
- Score tracking and persistence during play sessions
- Simple and clear user interface
- Responsive button interactions

## How to Play
- Click on the screen to increase your score
- [ESC] to open the shop
- Accumulate points to purchase from the shop
